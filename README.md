# Pomodoro App

## Preview

![](https://github.com/ardhii-m/pomodoro-app/public/preview.gif)

This Project is mainly to train my Tailwind CSS skills.

Framework used: 
- ReactJS
- Vite
- Tailwind CSS

## Features
- Timer, alternating between 25-minute and 5-minute timer.
- Task list
- Background Theme with context.
